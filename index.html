jQuery Plugin Demos


Flapper

filed under templates, internet

Flapper is a plugin that emulates the old Solari or split-flap display boards. This style of display used to be common in airports and train stations, and a few still have them. Flapper uses CSS to produce the effect.

Form Validation

filed under templates, internet

This is a form validation plugin from jqueryvalidation.org that will check what has been entered into your form.

Please provide your name, email address (won't be published) and a comment

Note how after you enter text into a field, it will immediate put up a message telling you that it did not check out. It really checks what is being entered. Try typing some garbage text into the email field. It can verify whether it is formatted like a proper email address.

FancyBox

Filed under templates, internet

This is FancyBox. It is an awesome jQuery plugin from fancyApps. FancyBox is a lightbox style plugin. There several different effects that you can add. Notice the elastic effect when the pictures are opened and closed. They do not merely appear on the screen - there is animation. Also try using your mouse scroll wheel, or click on the left or right side of the picture to transition between them.
